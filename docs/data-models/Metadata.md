# Metadata

Key             |Data Type     |Required  |Description
-------------   |---------     |------    |----
`id_doc`        |string        |auto      |unique database id
`log`           |list of dict  |auto      | version log
`permissions`   |list          |optional  |
`name`          |string        |required  |
`references`    |list          |optional  |
`tags`          |list          |optional  |
`notes`         |string        |optional  |
